# Library Inventory Manager

Simple library management system for Python lab.

## Files
- book.py - Book class
- inventory.py - Library inventory  
- library_inventory_manager.py - Main manager
- main.py - Menu interface
- books.json - Data file

## How to run
1. Save all files in one folder
2. Run: `python main.py`

## Features
- Add books with ISBN check
- Issue/return books
- Search by title
- Save to JSON file
